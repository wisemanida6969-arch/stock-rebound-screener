# 서버 연동 없이 모바일에서 실시간으로 보기

## 원리

- **백엔드·Streamlit·집 PC 서버 없음.**
- **폰 브라우저(또는 홈 화면에 넣은 PWA)** 가 Yahoo Finance 주소로 데이터를 요청합니다.
- 브라우저 보안 때문에 Yahoo는 직접 호출이 막힐 수 있어, **공개 CORS 프록시**(allorigins 등)만 경유합니다. 이것도 “당신이 서버를 돌리는 것”은 아닙니다.

## 바로 쓰는 방법

1. **`index.html`이 올라간 HTTPS 주소**를 폰에서 연다.  
   - 예: GitHub Pages → `https://아이디.github.io/저장소명/`
2. 맨 위 **티커 입력 → 「실시간 차트」**  
   - 장중이면 **1분봉**이 **15초마다** 갱신됩니다.  
   - 다른 앱으로 가면 갱신이 잠시 멈춥니다(배터리·데이터 절약).

## 홈 화면에 추가 (앱처럼)

- **iPhone**: Safari에서 공유 → **홈 화면에 추가**
- **Android**: Chrome 메뉴 → **홈 화면에 추가** / **앱 설치**

`manifest.json`으로 **전체 화면(standalone)** 으로 뜨게 해 두었습니다.
